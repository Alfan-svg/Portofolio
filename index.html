About Me

Halo! Saya adalah mahasiswa Informatika Semester 6 di Universitas Al Azhar Indonesia yang memiliki ketertarikan mendalam pada dunia teknologi, seperti pada bidang Machine Learning dan Cybersecurity. Saat ini saya sedang aktif mengeksplorasi implementasi kecerdasan buatan untuk analisis data kesehatan, serta mendalami teknik keamanan jaringan seperti analisis malware dan sistem deteksi intrusi (IDS/IPS). Saya percaya bahwa kombinasi antara keamanan data dan kecerdasan buatan adalah kunci teknologi masa depan. Selain itu, saya juga senang bereksperimen dengan pengembangan web modern menggunakan teknologi seperti Bootstrap untuk memvisualisasikan hasil analisis data saya.
